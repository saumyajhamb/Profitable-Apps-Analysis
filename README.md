*Profitable Apps Analysis – Apple App Store & Google Play Store (SQL EDA)*
_____________________________________________________________________________________________________________________________________________________________________________
Overview

This project focuses on identifying potentially profitable mobile app categories and trends by performing Exploratory Data Analysis using SQL on two major marketplaces — the Apple App Store and the Google Play Store. The analysis aims to extract meaningful business insights from raw app data by querying large datasets and comparing platform-wise performance indicators such as ratings, reviews, pricing, installs and category popularity.
_____________________________________________________________________________________________________________________________________________________________________________
Objective

1. Discover app categories with strong engagement and demand

2. Compare profitability indicators between iOS and Android ecosystems

3. Analyze the relationship between ratings, reviews, installs and price

4. Identify trends that can help developers make data-driven product decisions
______________________________________________________________________________________________________________________________________________________________________________
Datasets

1. Apple App Store Dataset

2. Google Play Store Dataset

Both datasets were analyzed separately and later compared to identify cross-platform patterns.
_______________________________________________________________________________________________________________________________________________________________________________
Tools & Technologies

1. SQL - Primary analysis tool

2. Jupyter Notebook for executing queries

3. Excel / CSV for raw data storage
__________________________________________________________________________________________________________________________________________________________________________________
Insights Generated

1. Categories with consistently high user engagement

2. Differences in pricing impact across platforms

3. Correlation between number of reviews and ratings

4. Market saturation trends in specific app genres

5. Platform-specific opportunities for new developers
__________________________________________________________________________________________________________________________________________________________________________________
Use Cases

1. App market research

2. Startup idea validation

3. Product strategy planning

Conclusion

This project demonstrates practical use of EDA to extract meaningful business insights from real-world datasets. By comparing two major app ecosystems, the analysis highlights trends that can guide smarter decision-making for developers, analysts and entrepreneurs entering the mobile app market.
